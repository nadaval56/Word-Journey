# 🧭 מסע מילה — Word Journey

כלי ווב חינוכי שמתחקה אחרי **מסעי הַשְׁאָלה אמיתיים ומתועדים** של מילים עבריות אל שפות אחרות:
לאן דלפה המילה, דרך אילו שפות־מתווכות, ואיזה שיבוש פונטי וסמנטי היא ספגה בכל תחנה —
עד לצורה הסופית, שלעיתים כמעט בלתי מזוהה (נכון, `samedi` הצרפתית היא **שַׁבָּת**).

מיועד לתלמידים מחוננים ולמורים. ‏100% צד־לקוח, Vanilla JS, בלי שלב build — רץ ישירות מ־GitHub Pages.

**המאגר כולל למעלה ממאה מילים מתועדות** (‏112 ערכים + 6 זוגות קוגנטים), הפרושׂות על כל מנגנוני ההַשְׁאָלה ועל מגוון רחב של שפות יעד: גרמנית ואנגלית (דרך יידיש ורוטוולש), הולנדית (בַּרגוּנס), הונגרית, רוסית (פֶניה), יוונית ולטינית, ספרדית ואיטלקית, ערבית, געז — ואפילו סינית (תעתיקים ליטורגיים בתרגומי המקרא הנוצריים, כמו 阿們 = אמן). כל ערך אומת מול מקורות אטימולוגיים (OED, Duden/DWDS, Etymonline, Etymologiebank ההולנדי, RAE, Leslau ועוד), וכל ערך לא־ודאי מסומן במפורש. כל המקורות מוצגים כקישורים.

**מילון מונחים מוקפץ:** מונחים בלשניים/היסטוריים שאולי אינם מוכרים (וולגטה, געז, רוטוולש, תרגום השבעים, פרוטו־שמית, קוגנט…) מסומנים אוטומטית בטקסט; לחיצה עליהם פותחת הסבר קצר — בלי לצאת לגוגל. המונחים וההסברים מוגדרים ב־`js/glossary.js`.

**⚖️ אינדקס חוקי הצליל:** לשונית ייעודית שהופכת את שינויי הצליל החוזרים (קָמָץ אשכנזי→o, שי"ן→s, עיצור גרוני נעלם ועוד — 10 חוקים) לאובייקטים ממשיים. לחיצה על חוק מציגה את **כל** המילים שמדגימות אותו — הקפיצה מאנקדוטה למערכת. כל תחנה מתויגת בחוקים שהיא מדגימה (`station.laws`), והחוקים מוגדרים ב־`meta.soundLaws`.

**🔊 שמיעת השחיקה:** כפתור 🔊 בכל תחנה מקריא את הצורה בשפת התחנה (דרך ה־TTS המובנה בדפדפן) — כדי לשמוע איך שַׁבָּת הופכת ל־shabes ול־samedi. ראו `js/audio.js`.

## המסגרת המושגית: הַשְׁאָלה ≠ קרבה (קראו לפני שמוסיפים ערכים!)

הכלי עוסק ב**הַשְׁאָלה (borrowing)** בלבד — מילה שעברה מדובר לדובר. הוא **אינו** עץ ירושה גנטי של שפות:

- **הַשְׁאָלה:** דוברי עברית הכניסו מילה עברית לשפה אחרת (מסחר, סלנג, תרגומי מקרא). יש לה מסלול ותחנות. ✔️ נכנסת למאגר כ"מסע".
- **קרבה/ירושה (cognate):** שתי מילים דומות כי שתי השפות צאצאיות של אב קדמון משותף. עברית וערבית הן **אחיות** (שתיהן מפרוטו־שמית) — ולכן `שלום/سلام` הוא קוגנט. ❌ **אסור** להציגו כמסע; מקומו ברשימת ה־`cognates`, וההבחנה מוסברת בעמוד "מה זה?".

דוגמת האב־טיפוס (געז): `ጸባዖת tsebaot` ("צבאות") היא **הַשְׁאָלה** — לגעז כבר הייתה מילה מקומית (`seraweet`) והעברית נכנסה ברובד הדתי. לעומתה `አማን aman` ("אמת") היא כנראה **קוגנט מולד**. אותה שפה, שני סיפורים.

## ארבעת מנגנוני ההַשְׁאָלה

| מנגנון | משמעות | דוגמה |
|---|---|---|
| `internal-jewish` | רובד עברי קבוע בשפה יהודית (יידיש, לאדינו…) | שבת ← shabbes |
| `leak-to-majority` | דליפה מיידיש לשפת הרוב, לרוב דרך רוטוולש | גנב ← Ganove ← ganef |
| `ancient-religious` | כניסה דרך תרגומי מקרא ופולחן | צבאות ← tsebaot (געז) |

## תיוג ודאות — המנדט האתי

אטימולוגיה מוצפת מיתוסים. **כל ערך חייב תג ודאות ומקורות**:

| תג | תווית | מתי |
|---|---|---|
| `documented` 🟢 | מתועד | קונצנזוס מחקרי / מילונים אטימולוגיים |
| `probable` 🟡 | משוער | תמיכה סבירה, לא ודאי |
| `debated` 🟠 | שנוי במחלוקת | אטימולוגיות מתחרות (למשל `Kaff`: עברית כפר מול רומאני gav) |
| `folk` 🔴 | עממי — נדחה | מיתוס נפוץ שנדחה; נכלל **בכוונה** עם הפרכה, כפינת "שוברי מיתוסים" |

ה־UI לעולם לא מציג ערך `folk` בלי אזהרה בולטת.

## איך מוסיפים ערך חדש

1. **אמתו מול מקור אטימולוגי רציני** (OED, Duden/DWDS, TLFi, Klein, Leslau, EHLL…). בלי מקור — הערך לא נכנס. עדיף מאגר קטן ומדויק ממאגר גדול ומפוקפק.
2. הוסיפו אובייקט ל־`words` ב־`data/words.json`:

```jsonc
{
  "id": "gannav",                    // מזהה לטיני ייחודי (משמש ב-URL)
  "hebrew": "גַּנָּב",
  "translit": "gannav",
  "meaning": "גנב",
  "mechanism": "leak-to-majority",   // המנגנון הראשי
  "certainty": "documented",         // ראו טבלת התיוג!
  "sources": ["Rendsburg (EHLL)", "Duden – Ganove"],
  "note": "הערת העשרה שתוצג בעמוד המילה (רשות)",
  "journeys": [                      // מסלול אחד או יותר (מילה מסועפת ← עץ)
    {
      "title": "אל הגרמנית — Ganove",
      "mechanism": "leak-to-majority",  // רשות: עוקף את המנגנון הראשי למסלול זה
      "path": [
        { "lang": "עברית", "form": "גַּנָּב", "translit": "gannav", "meaning": "גנב", "note": "מקור" },
        { "lang": "יידיש", "form": "גנבֿ", "translit": "ganev", "meaning": "גנב",
          "change": "שכבה עברית ביידיש; בי\"ת רפה נהגית v" }
      ]
    }
  ]
}
```

- `change` בכל תחנה = הסבר קצר של השינוי הפונטי/סמנטי מהתחנה הקודמת. **זה הבשר החינוכי** — אל תשאירו ריק בלי סיבה.
- לערכי `folk` חובה להוסיף `folkClaim` (הטענה) ו־`debunk` (למה נדחתה).
- קוגנטים נכנסים למערך `cognates`, לא ל־`words`.

3. ודאו שה־JSON תקין (`python3 -m json.tool data/words.json`) ובדקו בדפדפן.

## מגבלות טכניות (מכוונות)

- Vanilla JS בלבד, `<script type="module">`, בלי framework ובלי build.
- כל הנתונים בקובץ JSON סטטי; אין שרת.
- ממשק עברית מלא, RTL, מותאם לטאבלט ולמובייל (יעדי מגע ≥ 44px; רצועת המסע אנכית במסכים צרים).
